# CONTEXT

## Project Overview
**Auratekk** is a high-end, futuristic portfolio and service showcase website designed with a dual-theme aesthetic (Dark/Light modes). It emphasizes visual impact through advanced animations, 3D elements, and modern UI/UX principles.

## Core Value Proposition
- **Immersive Experience:** Utilization of 3D models (Spline), scroll-triggered animations (GSAP/ScrollTrigger), and dynamic cursor effects.
- **Dual Personality:** A "Midnight" dark mode for a cinematic feel and a "Daylight" light mode for clarity and professionalism.
- **Service Showcase:** Clearly defined service offerings (Startup Acceleration, Data Science, AI Solutions, Cloud Infrastructure) presented with interactive cards.
- **Lead Generation:** Integrated contact forms connecting to Supabase for real-time data capture.

## Target Audience
- Clients seeking premium digital transformation services.
- Tech startups looking for acceleration and infrastructure partners.
- Enterprises needing scalable cloud and AI solutions.

## Current Status
- **Frontend:** Fully functional HTML/CSS/JS prototype with Tailwind CSS.
- **Backend:** Supabase integration for form submissions.
- **Deployment:** Static hosting ready.
